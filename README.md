# AI-Powered Personal Finance Intelligence Platform

A machine learning-based personal finance application that helps users track expenses, analyze spending behaviour, forecast future expenses, detect unusual transactions, and receive personalized financial recommendations.

---

## Features

###  User Authentication
- User signup and login
- Secure session handling
- User-specific financial data storage

###  Expense & Income Tracking
- Add income transactions
- Add daily expenses
- Categorize spending
- Track transaction history

###  Interactive Dashboard
- Total income overview
- Expense analysis
- Savings calculation
- Category-wise spending visualization
- Monthly expense trends

###  AI-Powered Features

#### Expense Forecasting
- Predicts future spending patterns using machine learning
- Implemented using Linear Regression

#### Anomaly Detection
- Identifies unusual spending behaviour
- Uses Isolation Forest algorithm

#### Financial Insights
- Generates personalized spending insights
- Provides budgeting suggestions

###  Financial Planning
- Savings goal tracking
- Financial Health Score
- Budget monitoring

###  Reports & Export
- Download transaction data as CSV
- Generate monthly financial reports as PDF
